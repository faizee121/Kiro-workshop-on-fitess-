# Kiro Workshop on Fitness Center Management System

## Overview

This workshop demonstrates how to build a Fitness Center Management Application using:

- Kiro IDE
- Amazon DocumentDB
- Python Flask
- PyMongo
- Bootstrap

## Features

- Client Registration
- Workout Management
- Body Measurements
- Progress Analytics
- Authentication
- Reporting

## Architecture

Browser
→ Flask
→ Business Logic
→ PyMongo
→ Amazon DocumentDB

## MVP Tasks

1. Project Structure
2. DocumentDB Connection
3. Client Management
4. Workout Programs
5. Body Measurements
6. Dashboard
7. Authentication
8. Web Interface
